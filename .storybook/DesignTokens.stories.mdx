# Neural Pulse Design Tokens

Design tokens are the visual design atoms of the design system â€” specifically, they are named entities that store visual design attributes. We use them in place of hard-coded values to maintain a scalable and consistent visual system for UI development.

## Colors

Color tokens are used to ensure consistent color usage across all components and layouts.
