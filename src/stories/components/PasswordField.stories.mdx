# PasswordField

`PasswordField` is a component for inputting passwords with an option to toggle visibility.

## Usage

```tsx
<PasswordField id="user-password" name="password" placeholder="Enter your password" onChange={event => console.log(event.target.value)} />

import React, { useState } from 'react';

<Canvas>
  <Story name="Default">
    {() => {
      const [password, setPassword] = useState('');

      const handleChange = event => {
        setPassword(event.target.value);
      };

      return (
        <PasswordField
          id="user-password"
          name="password"
          placeholder="Enter your password"
          value={password}
          onChange={handleChange}
        />
      );
    }}
  </Story>
</Canvas>
